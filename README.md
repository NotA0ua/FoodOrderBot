## Before installation
**Don't forget** to set .env variables

## Installation
### Linux/MacOS
```
python -m venv .venv
source ./.venv/bin/activate
pip install -r requirements.txt
python -m app
```

### Windows
```
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
python -m app
```
